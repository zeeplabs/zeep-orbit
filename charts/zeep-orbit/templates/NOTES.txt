zeep-orbit instalado com sucesso!

Para acessar a API dentro do cluster:

  kubectl port-forward svc/{{ include "zeep-orbit.fullname" . }} 8080:80 -n {{ .Release.Namespace }}

Endpoints disponíveis:
  GET  /health          — health check
  GET  /metrics         — Prometheus metrics
  GET  /docs/           — lista de apps (Swagger UI)
  GET  /docs/{app}      — Swagger UI do app
  GET  /{app}/{table}/  — CRUD (requer JWT Bearer)

Para verificar os pods:

  kubectl get pods -l app.kubernetes.io/name={{ include "zeep-orbit.name" . }} -n {{ .Release.Namespace }}

Para ver os logs:

  kubectl logs -l app.kubernetes.io/name={{ include "zeep-orbit.name" . }} -n {{ .Release.Namespace }} --follow
